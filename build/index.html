技能 Skill

Core 核心技術

HTML

熟練 HTML5 語意化結構，建立清晰且具邏輯的頁面骨架。

CSS / SCSS

運用 SCSS 管理樣式結構與層級，搭配 Bootstrap 建構穩定且具擴展性的響應式介面，確保網頁在各種裝置上皆有良好顯示效果。

JavaScript

熟悉 ES6+ 語法，能處理複雜資料邏輯，提升程式碼可讀性與維護性。

React

具備 SPA 架構開發經驗，透過組件化思維建立可重用 UI 模組；熟悉 React Hook Form 表單驗證與 Redux Toolkit 狀態管理，維持複雜 UI 的資料流清晰與可預測性。

Tools 開發工具

Vite

熟悉現代化前端建構工具，能快速建立開發環境並優化專案打包與開發流程。

Mock API

使用 JSON Server 建立 Mock API 環境，支援前後端分離開發流程。

API Integration

可使用 Axios 串接 RESTful API，處理資料請求、錯誤處理與前端狀態回饋。

Collaboration 團隊協作

Code Quality

透過 ESLint 建立一致的程式碼風格與品質標準，提升團隊協作效率。

Git / GitHub

熟悉 Git Flow、分支管理與 Pull Request 協作流程，確保專案開發紀錄清晰。

AI Collaboration

探索 AI Agent 協作開發流程，利用 AI 工具輔助需求拆解、除錯與開發效率提升。
